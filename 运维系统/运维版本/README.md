>>  可通过两种方式连接数据库：

        (CONNECT_DATA =
        (SERVICE_NAME = orcl)
        )
    ) `   
>>> 然后在登录界面的配置功能，通过填写`数据库`项，格式：oracle服务名称。如上示例，应填入：orcl188，即可。
